Purpose

CycleFit Planner is a web application that helps recreational cyclists create personalized cycling training plans based on their fitness level, goals, and riding preferences. The application calculates BMI, estimates calories burned, recommends weekly training volume, and stores personalized plans using Local Storage.

Audience

The intended audience is recreational and amateur cyclists between 18 and 65 years old who want a free, easy-to-use tool for organizing their cycling workouts and monitoring their progress.

Dynamic Elements

  • Interactive form with validation
  • BMI calculation
  • Calories burned estimation
  • Personalized weekly training plan
  • Hydration and nutrition recommendations
  • Automatic DOM updates
  • Save, edit and delete plans with Local Storage

Pseudocode

START
Load saved plans
IF plans exist
 Display plans
ENDIF

User submits form
 Validate inputs
 IF invalid
   Show error
 ELSE
   Calculate BMI
   Calculate calories
   Generate training plan
   Display recommendations
   Save to Local Storage
 ENDIF

User edits or deletes plan
 Update Local Storage
 Refresh page
END
