Overview

Purpose

CycleFit Planner is a web application that helps cyclists create a personalized cycling training plan based on their personal information, fitness goals, and riding preferences. The application estimates calories burned, weekly training volume, and expected progress while providing recommendations to improve cycling performance safely.

Audience

The intended audience is recreational and amateur cyclists between the ages of 18 and 65 who want a simple way to organize their training without using expensive fitness applications. It is also useful for beginners who are learning how to plan their cycling workouts.

Dynamic elements

The application will use JavaScript to create an interactive experience for cyclists. Users will enter their personal information, choose a cycling goal, and receive a customized training plan.

  • Collect cyclist information through an interactive form.
  • Validate required fields before generating a plan.
  • Calculate Body Mass Index (BMI).
  • Estimate calories burned based on weight, ride duration, and average speed.
  • Generate a personalized cycling training plan based on the selected goal.
  • Display recommended weekly distance and riding schedule.
  • Show hydration and nutrition recommendations.
  • Allow users to save their cycling plan using Local Storage.
  • Allow users to edit or delete saved plans.
  • Update the results instantly whenever the user changes the input values.
